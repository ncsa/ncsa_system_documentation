.. _scrontab:

Recurring Jobs (scrontab)
===========================

.. warning:
   ``scrontab`` is not enabled on all clusters. You can check if ``scrontab`` is enabled by running ``scrontab -l`` on a **compute node**.

Use ``scrontab`` to manage Slurm ``crontab`` files, which allow you to schedule recurring batch jobs. Note that ``scrontab`` jobs are subject to queueing delays and are not guaranteed to start at the preferred times.

See the `scrontab Slurm documentation <https://slurm.schedmd.com/scrontab.html>`_ or man page (``man scrontab``) for more inforation.
